# JHSC Worker Hub

A worker-side tool for one Joint Health & Safety Committee co-chair and their rep team. Built to support statutory functions under Ontario OHSA and the Canada Labour Code Part II — minutes, action items, hazards, recommendations, evidence, and inspections.

Single-tenant. Single-workplace. Non-commercial. Designed for the floor, freezer, meeting room, and commute — mobile-primary with desktop as the expanded view. Currently in active development on Release 1 (Milestone 1.1 — Foundation).

This is personal mission-critical tooling for one rep team, not a product. The repository is public for transparency and as reference for any rep team that finds it useful; there is no expectation of contribution, and no support is offered.
